# HA XO MQTT Updater Add-on

Repozitář obsahuje Home Assistant Add-on, který čte metriky z XCP-ng hosta přes XO API a publikuje je do MQTT s podporou MQTT Discovery.

Instalace:
- Přidejte repozitář URL do HA Supervisor → Add-on Store → Repositories
- Nainstalujte add-on ze složky `xo_mqtt_updater`
